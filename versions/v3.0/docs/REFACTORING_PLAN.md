    """–ò–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –æ —Ä–∞–Ω–≥–µ –∏ —É—Ä–æ–≤–Ω–µ."""

            xp_progress = f"\n–î–æ —Å–ª–µ–¥—É—é—â–µ–≥–æ —É—Ä–æ–≤–Ω—è: {xp_to_next} XP"
